How To run ?

1. git clone https://github.com/ycandramonanza/Perpustakaan-APP.git 
2. composer install
3. npm install 
4. php artisan key:generate
5. buat file .env dan konfigurasi database
6. php artisan config:clear
6. php artisan migrate
7. php artisan db:seed
8. php artisan serve
